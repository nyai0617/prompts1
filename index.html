Geppo導入事例探すくん・検索プロンプト集

はじめに

事例探すくんにGeppoの事例を読み込ませて、欲しい情報やヒントをサクッと引き出すための「検索プロンプト（質問）」を集めました！

本文書に収録されたクエリは、基本的な情報探しから、事例の裏側にあるストーリーや成功の秘訣を探るような深掘りまで、いろんな調査に対応できるように作っています！

カンタンな使い方:

  1. このリストの中から自分の目的にピッタリなプロンプトを選んでコピーする。それぞれのプロンプトには「こんなことが知りたい時におすすめ！」っていう狙いがあるから、それを参考に選ぶとグッド！
  2. 最後に、コピーしたプロンプトをAIアシスタントの質問欄にペーストして、検索を実行するだけ！出てきた答えは、あくまでAIが事例を元に作ったものだから、「なるほど、こんな見方もあるのか〜」くらいの気持ちで、参考にするのがおすすめです。

第一部：【基本編】まずはコレ！目的の事例をサクッと見つけよう

この章では、いろんな会社の状況をざっくり把握するための、基本的な探し方をまとめました。これを使えば、たくさんの事例の中から、自分たちと似たケースをすぐに見つけられるはずです！

第1章：「こんな目的！」で探す

似たような課題で悩んでる会社の事例を参考にしたい時に、すごく便利！

1.1 従業員の定着・離職防止に取り組んでる事例、知りたい！

# 命令
ソース内で言及される企業のうち、「離職防止」「定着率向上」「リテンション」を目的としてGeppoを活用した事例を網羅的に列挙すること。

# 条件
・該当する企業名
・導入前の課題
・具体的な活用方法
・それによって得られた成果または定性的な効果
上記4項目が識別できるよう、本文から関連箇所を引用しつつ要約を生成すること。出力は、企業ごとに項目を立て、箇条書き形式で整理すること。

1.2 新人や中途社員がすぐ辞めちゃう…何かヒントになる事例ある？

# 命令
「早期離職」または「オンボーディング」の課題解決を目的としてGeppoを活用した事例を抽出すること。本分析は、人材の初期定着段階における阻害要因を特定し、介入策の有効性を評価する上で極めて重要である。

# 条件
特に「新卒」または「中途入社者」に対し、いかなる支援（例：面談、フォロー体制）が提供されたかが具体的に詳述されている箇所を特定されたい。また、人事部門と現場部門間の連携体制に関する言及が含まれる場合は、当該情報も併せて抽出すること。

1.3 エンゲージメントが上がった！っていう会社の事例が見たいな

# 命令
「エンゲージメント向上」または「従業員満足度（ES）向上」を主要目的として掲げる事例を特定されたい。

# 条件
サーベイ結果が組織改善活動にどのように統合され、PDCAサイクルが運用されたか、そのプロセスが理解できるよう説明を生成すること。「エンゲージメントスコア」が重要業績評価指標（KPI）として言及されている場合は、その部分を詳細に引用し、可能であればその定義についても言及すること。

第2章：「この施策！」で探す

Geppoが、他の人事施策とどう組み合わさって、いい効果を出しているのかを知りたい時に使ってください！

2.1 1on1や面談がもっと良くなった！っていう事例は？

# 命令
Geppoを「1on1ミーティング」または「上長との面談」に応用している事例を検索すること。

# 条件
Geppoの導入が、1on1ミーティングの「質」に対し、いかなる影響を及ぼしたか（例：対話の触媒、議題の明確化、心理的開示の促進など）が示されている記述を引用・要約すること。可能な限り、導入前後のコミュニケーションの質的変化に関する直接的な言及を引用すること。

2.2 「答えっぱなし」にしない！回答者へのフィードバックを工夫してる事例は？

# 命令
従業員からの回答に対し、「答えっぱなし」にせず、会社や人事がどのようにフィードバックしているか、具体的な取り組みをしている事例を抽出してください。

# 条件
・全社向けの説明会やイントラネットでの結果サマリー報告
・「こんな意見を元に、ここを変えました」という改善報告
・ポジティブなコメントへの感謝の伝達
など、どんな方法で「答えてくれてありがとう、ちゃんと見てるよ」という姿勢を示しているか、その工夫を教えて。

2.3 良いコメント、どう活かしてる？朝会や会議でシェアしてる事例は？

# 命令
Geppoに寄せられたポジティブなコメントや、改善に繋がる建設的な意見を、組織内で共有（シェア）している事例を探してください。

# 条件
・朝会、週次定例、全社集会など、どのような「場」で共有しているか。
・「良い行動の称賛」「課題の共通認識化」「改善アイデアの募集」など、どのような「目的」でコメントをシェアしているか。
その具体的なシーンがわかるエピソードを教えて。

第二部：【上級編】もっと深掘り！「すごい発見」を引き出そう

ただ事例をなぞるだけじゃ物足りない！その裏にある本当のストーリーや、組織を動かすような大きなヒントを見つけ出したい、探求心旺盛なあなた様へ。

第1項：「失敗から学んだ」リアルな話、教えて！

（成功事例の再現性を高めるだけじゃなく、隠れたリスクを事前に知って、自分たちの計画をより良くするためのヒントが満載！）

# 命令
導入事例の中から、「導入初期段階における非効率性」「形骸化の兆候」「試行錯誤の過程」といった、何らかの"失敗"または"困難"を克服したことが示唆される事例を検索すること。本分析は、成功事例の再現性を高めるだけでなく、潜在的リスクを予見し、導入計画の堅牢性を確保するための重要な示唆を提供する。

# 条件
・直面した「失敗」または「困難」の具体的性質。
・当該状況を打開するために講じられた具体的施策。
・当該経験から得られた「教訓」または「知見」が示されている部分を引用し、要約すること。

第2項：「え、そんな効果も！？」っていう想定外のラッキーな話

（狙っていなかったけど、思わぬプラスの効果があった事例を発見！ツールの新しい価値や、意外な使い方のヒントが見つかるかも！）

# 命令
Geppo導入事例の中から、「当初の目的」とは異なる「非意図的な成果」または「偶発的な副産物」が創出されたと解釈しうる事例を抽出すること。組織施策がもたらす予期せぬ正の外部性を特定することは、ツールの価値を再定義し、新たな活用可能性を拓く上で有益である。

# 条件
・企業が初期に設定した「導入目的」の特定。
・運用過程で発現した「非意-図的成果」の特定。
・当該の非意dto的成果が発現した背景またはプロセスに関する記述を引用・要約すること。

第3項：「ウチはあえてこうしてる！」常識を覆す使い方

（「普通はこうだよね」っていうセオリーをあえて無視！自分たちの会社にピッタリな、オリジナルの運用方法を考える刺激が欲しい時に！）

# 命令
Geppoの運用方法において、一般的な理論や推奨される手法とは異なる、"独自の工夫"または"特異な規則"を設けている事例を抽出すること。本クエリは、ベストプラクティスとされるものに対する批判的検討を促し、各組織の固有の文脈に最適化された運用モデルを構想するための知的刺激を提供することを目的とする。

# 条件
・当該企業が実践している「独自の運用方法」の特定。
・そのような非標準的運用を採用する「目的」または「思想」の説明。
・当該の特異な運用によって得られている効果の提示。

第4項：「経営トップが動いた！」インパクト絶大な話

（人事データが、会社全体の戦略を動かすほどの力を持った事例を発見！人事の仕事の価値を、もっと大きな視点で証明したい時に！）

# 命令
従業員のデータまたはコメントが、人事領域を超越し、「経営判断」「事業戦略の再検討」「新規サービスの開発」といった、組織の根幹に関わる意思決定に影響を及ぼした事例の有無を調査すること。本分析は、人事データが戦略的資源として経営に活用される高度な段階を示す事例を特定し、HR機能の戦略的価値を論証する上での根拠を提供する。

# 条件
・従業員から提出された「意見・情報」の性質。
・それが関連付けられた「経営レベルの意思決定」の内容。
・その結果として組織または事業に及んだ影響。
以上の因果関係を再構成して提示すること。

第5項：「ある一人の人生が変わった」感動的なエピソード

（会社の成果っていう大きな話だけじゃなく、一人の社員の人生に寄り添った、人間味あふれるストーリーに触れたい時に！）

# 命令
Geppoが契機となり、一従業員の「キャリア」「就業形態」、あるいは「人生」に好影響をもたらす転機が訪れたことを示す、個人的なエピソードまたは物語が含まれる事例を検索すること。本分析は、マクロな組織的成果の背後にある、ミクロな個人的体験の重要性を浮き彫りにし、施策の人間的側面に対する理解を深めることを目的とする。

# 条件
・当該従業員によって発信された「意見」の内容。
・それを受理した組織または上長によって実行された「対応」。
・その結果として当該従業員の状況にもたらされた好転の内容。
組織の成功事例という観点ではなく、「一個人の物語」として記述されている部分を抽出すること。
